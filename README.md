# CodeAlpha_tasks
This repository contains a collection of beginner-friendly Python mini projects developed to strengthen core programming skills and demonstrate practical problem-solving. The projects include a Hangman Game, Stock Portfolio Tracker, and AI Chatbot, covering essential Python concepts such as variables, data types, loops, conditional statements, functions, dictionaries, lists, string manipulation, input validation, exception handling, file handling, and modular programming. Each project is designed with clean, well-commented, and organized code, making it suitable for beginners, students, and anyone looking to practice Python through real-world applications. These projects can be run directly from the command line without requiring any external libraries or additional setup.
